# How Mybatis takes parameters from frontend

### I used
 - MySQL 8.0
 - DataGrip
 - Mybatis 3.5

### Parameters
 - primitive types / more than one primitive types
    - `byte`
    - `short`
    - `char`
    - `int`
    - `long`
    - `float`
    - `double`
    - `boolean`
  - reference types
    - `Object`
    - `List`
    - `Map`
    - etc
